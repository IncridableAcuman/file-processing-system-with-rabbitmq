## File Processing System with Java Spring Boot Gradle and Message Broker
** *
### Overview:
Bu loyihada java spring boot gradle va message broker orqali file yuklash jarayonini ishlab chiqdim.
Message broker bo'lgan RabbitMQ orqali Async ma'lumotlar almashinuv jarayonini ko'rib chiqdim.
** *

### Technologies
* Java
* Spring Boot
* RabbitMQ
* PostgreSQL
* Docker
** *
